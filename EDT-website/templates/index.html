About the project


Customers want to be reasurred as to when there packages will arrive. Through the use of machine learning this is doable as we are able to predict to a high degree when a customers package will arrive !


This project utalises HTML5, CSS, Python, SAS and data science related packages such as Pandas, SciKit learn, matplotlib and numpy


The website currently is built on Flask which is a Python web framework. The two models i have created, Gradient Boosting and Random forest are utalised through an API which send requests (the values for prediction) to each individual model, then POST's the prediction back to the home page.
